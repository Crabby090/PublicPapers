# Papers

Each paper lives in `papers/<year>/<slug>/`.

Expected files
- `metadata.yml` - paper metadata and status
- `outline.md` - research question and section plan
- `paper.md` - main manuscript with citations
- `sources.md` - full references list
- `claims.csv` - evidence log mapping claims to sources
- `notes.md` - research log and decisions
- `checklist.md` - pre-publish review list
- `data/` - datasets or extracted tables
- `figures/` - charts or images
- `exports/` - rendered outputs (pdf, docx)

Use `scripts/new-paper.ps1` to scaffold a new paper folder.
